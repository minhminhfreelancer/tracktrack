TrackTrack API Service

This is the API service for the TrackTrack analytics platform. This service handles data collection and processing for the analytics dashboard.

Available Endpoints

POST /api/track

Endpoint for collecting tracking data from websites.

Usage: This endpoint is called automatically by the tracking script embedded in websites.

Integration

To integrate tracking on your website, visit the script generator in your dashboard.

Documentation

For more information about the API and how to use it, please refer to the documentation.
